Playground

Instructions

This playground allows you to interact with network directly. The top-patch-intialized gradient ascent visualization (left) is shown. Select a unit in the specified neural network and layer, then draw on the canvas. The content of the canvas will be fed into the network, and the response of the unit will be displayed. Note: Vgg16 and ResNet18 are not fully supported.
